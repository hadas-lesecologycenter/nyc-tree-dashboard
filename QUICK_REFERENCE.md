# Event Dashboard - Quick Reference Card

## 🚀 Getting Started (30 seconds)

1. **Open**: `event-dashboard.html` in your browser
2. **Create**: Click event type + fill form + submit
3. **Track**: Click event → check off tasks as you go
4. **Done**: Progress saves automatically!

- **Share between browsers**: No, local only

## ❓ FAQ

**Q: Do I still use the spreadsheets?**
A: Yes! Dashboard reminds you of tasks, you manage execution in your current tools.

**Q: Will I lose data if I clear my browser?**
A: Yes. Phase 2 will add cloud backup. For now, be careful with browser clearing.

**Q: Can multiple people use this?**
A: Right now, each browser has its own data. Phase 2 adds cloud + multi-user.

**Q: Does it work offline?**
A: Yes! It works completely offline. Just open the file in your browser.

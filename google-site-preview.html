Once you deploy the Apps Script as a web app, the full dashboard will appear here.

To deploy the dashboard:

  1. Go to script.google.com
  2. Create a New Project
  3. Add Code.gs and Index.html from the repository
  4. Click Deploy → New deployment → Web app
  5. Set Execute as: Me, Access: Anyone in your org
  6. Copy the deployment URL
  7. Replace the iframe src below with your URL

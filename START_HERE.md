## 🚀 Quick Start (5 Minutes)

### Step 1: Open the Dashboard (30 seconds)
1. Find the file `event-dashboard.html` 
2. Double-click it to open in your browser
3. Bookmark the page for easy access

### Step 2: Read the Quick Reference (2 minutes)
Open and skim `QUICK_REFERENCE.md` to understand:
- How to create events
- What each event type checklist includes
- Where to find email templates

### Step 3: Create a Test Event (2 minutes)
1. Click an event type (e.g., "Field Trip")
2. Fill in details (name, date, location, coordinator, size)
3. Click "Create Event"
4. Click the event in the list to see the timeline
5. Check off a task to see progress update

**Done!** You now understand how it works.
